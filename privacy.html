Privacy Policy

This Privacy Policy explains how the Twitch Announcer Bot handles data.

Data Collected

The bot only processes Twitch API data (stream status, title, game) and Discord channel IDs. It does not collect personal user data.

Storage

No personal information is stored. Configuration values (tokens, IDs) are kept securely in environment variables.

Third Parties

The bot uses Twitch’s official API and Discord’s API. Both services have their own privacy policies.
